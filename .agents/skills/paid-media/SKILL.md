# paid-media Skill

## Description
The paid-media skill provides specialized workflows for AMADS agents to execute domain-specific tasks efficiently.

## When to Use
- When a campaign requires paid-media capabilities
- When an agent needs to execute a paid-media workflow
- When quality standards for paid-media must be met

## Steps
1. Receive task brief with objectives
2. Gather necessary data and context
3. Execute domain-specific workflow
4. Validate output against quality standards
5. Deliver results to requesting agent

## Tools Required
- Relevant MCPs and APIs as defined in agent configuration
- Access to shared knowledge base
- PostgreSQL for data persistence

## Output Format
- Structured deliverable matching task requirements
- Quality score and validation notes
- Metadata for tracking and analytics

## Examples
- Example 1: Research brief for new market entry
- Example 2: Content piece for social media campaign
- Example 3: Analytics report for weekly review
